# Uputstvo

### Pokretanje
```bash
python maze_generator.py        -> generise the_maze.json
python main.py                  -> ucitava the_maze.json i pokrece pretragu
```


### Datoteke
```
├── graph.py                    -> Implementacije algoritama pretrage
├── main.py                     -> Pokretanje algoritama za resavanje.
├── maze_generator.py           -> Podesavanje za lavirint.
├── README.md
├── the_maze.json               -> Datoteka generisana od strane skripta 'maze_generator'.
└── visualize_maze_optional
    ├── draw.sh                 -> Generise .png sliku od 'the_maze.json'.
    └── visualize_maze.py       -> Koristi se pri generisanju slika grafa.
```
